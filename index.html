The Box Model

Introduction: The Box Model

'The Box Model' refers to the rectangular boxes that are generated around the content they contain. The Box model incudes: content, padding, border and margin.


Padding

Padding is the inner most section of the box model. By using the 'padding' property, it is possible to set the padding dimensions. There can be 1-4 values with pixel values and percentage values (with percentage values refering to the width of the content block)

4 values

padding: 6px 11px 4px 9px;

changes the padding to 6px on the top, 11px on the right, 4px on the bottom and 9px on the left.

3 values

padding: 5px 10px 20px;

This Changes the padding to 10px on the left and right but 5px on top and 20px on the bottom.

2 values

padding: 5px 30px;

changes the padding to 5px on the top and bottom and 30px on the left and right.


  • To change the padding sides independantly we use the properties; padding-left, padding-right, padding-top, padding-bottom
  • values for padding values cannot be negative. percentage values for padding properties refer to the width of the generated box's containing block

Border

The Border is outside of the padding and the border property controls width, color and style.

Border shorthand rule

border: thick solid red;

This changes the border to a solid thick red line and will display like this in the browser


  • Specific properties include: border-top, border-top-style, border-top-width, border-top-color. Change top for left, right or bottom depending on the side you want to change

Margin

Margin is outside of the border and the margin property controls the margins width

Margin rule

margin: 2em;

This sets the margin width to 2em (200%) on all four sides.


  • negative values are allowed
  • percentage values refer to the width of the content box
